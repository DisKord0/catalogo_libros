# catalogo_libros
Proyecto Catálogo de Libros desarrollado en equipo con PHP. Usamos el patrón MVC para estructurar el código y MySQL para la base de datos. Implementamos el CRUD (Crear, Leer, Actualizar, Eliminar) para la gestión de la colección y un sistema de autenticación de usuarios para controlar el acceso. Enfocados en la modularidad y el trabajo colaborativo
